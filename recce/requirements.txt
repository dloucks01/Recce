# recce has NO third-party Python packages to install - it uses only the Python
# standard library, so it runs on an airgapped box with a stock Python 3.9+.
# (The .xlsx and .docx writers are hand-rolled; there is nothing to pip install.)
#
# It orchestrates these SYSTEM tools when present (all standard on Kali). Only
# nmap is required; every other tool is optional and its phase degrades cleanly
# (with a logged note) when the tool is absent:
#
#   nmap           (required) - scanning, service/version/OS detection, NSE vuln
#   masscan        (optional) - fast --fast network-wide port sweep
#   searchsploit   (optional) - offline exploit mapping (exploitdb package)
#   ldapsearch     (optional) - credentialed AD LDAP enumeration (ldap-utils)
#   netexec / nxc  (optional) - credentialed SMB/AD enum (credenum phase)
#   impacket       (optional) - Kerberoast / AS-REP / secretsdump (credenum phase)
#   ssh (+ sshpass)(optional) - credentialed Linux local checks (credenum phase)
#   chromium       (optional) - auto web screenshots in finding write-ups
#                               (or point RECCE_BROWSER at any Chrome/Chromium)
#
# Run `recce doctor` on the laptop to confirm which capabilities are available.
